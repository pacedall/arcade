# Pacedall Arcade 🕹️

A retro-modern arcade gaming platform built by **Pacedall Labs**.

## Structure

```
pacedall-arcade/
├── index.html        # Main homepage
├── login.html        # Login page
├── signup.html       # Sign up / registration page
├── styles.css        # All styles (CSS variables, responsive)
├── script.js         # Game data, leaderboard, mini snake demo, interactions
├── images/
│   └── logo.png      # Pacedall Arcade logo
└── README.md
```

## Features

- 🎮 **Featured Games Grid** — showcases hot and new releases
- 🗂 **Game Categories** — Action, Puzzle, Racing, Endless, Retro, Chill
- 🏆 **Leaderboard** — tabbed rankings across games
- 👑 **Pro Plan CTA** — monthly/annual pricing cards
- 🕹️ **Live Mini Snake Demo** — animated canvas game in the hero
- 📱 **Fully Responsive** — mobile, tablet, desktop
- 🌐 **Auth Pages** — login.html + signup.html

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts: Orbitron + Bebas Neue + DM Sans
- Canvas API for mini-game demo

## Deployment

Works as a static site — deploy to any static host:
- **GitHub Pages**: push to `main`, enable Pages in repo settings
- **Netlify / Vercel**: drag-and-drop the folder
